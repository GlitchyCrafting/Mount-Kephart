A Bit About The Design

UI

I chose the Dracula Colorscheme because I like it and it is easy on the eyes. I designed a clean, minimal interface because a bunch of seperators and stuff isn't needed for a site like this. Fonts are Space Mono for headers and Fira Mono for everything else.

Backend

I use the Crow Framework for defining routes and templates and the SqliteModernCpp Library to interact with the database because I, unlike others, like C++. For a while, though, I was tempted to switch to Django for Python due to some issues with a different Sqlite library, but as you can see I found a better library.
